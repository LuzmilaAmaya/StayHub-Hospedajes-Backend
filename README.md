🏨 StayHub - Sistema de Gestión de Hospedajes

Desarrollada para el sistema StayHub, una plataforma de gestión de hospedajes que permite administrar habitaciones y reservas.

Este backend está construido con una arquitectura por capas, siguiendo buenas prácticas de separación de responsabilidades y organización modular.

🚀 Tecnologías utilizadas

* Node.js

* Express

* MongoDB

* Mongoose

* Dotenv

* Nodemon
